# 个人博客示范项目

## 目录说明

- `_posts/`：博客文章目录
- `images/`：博客图片目录
- `_layouts/jakfeo.html`：自定义布局文件
- `assets.css`：全局样式
- `index.md`、`about.md`：页面文件，均使用 `jakfeo` 布局
- `_config.yml`：Jekyll 配置

## 使用说明

1. 修改 `_config.yml` 中站点信息  
2. 添加文章到 `_posts/`  
3. 上传图片到 `images/`  
4. 自定义样式写在 `assets.css`  
5. 页面指定布局为 `jakfeo`
